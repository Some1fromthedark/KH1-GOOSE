# KH1 GOOSE

**Game Object Overhaul System Enhancement**

KH1 GOOSE is a memory expansion and engine extension framework for *Kingdom Hearts 1 Final Mix (PC)*.
It enables advanced modding capabilities by removing or extending many of the original engine’s hard limitations.

---

## 🧠 Overview

The original KH1 engine was built with strict assumptions about memory usage, asset sizes, and object limits. These constraints make certain types of mods difficult or outright impossible.

**GOOSE** addresses these limitations by:

* Expanding available memory regions
* Increasing object and asset limits
* Allowing larger models, animations, and data structures
* Providing a foundation for more complex gameplay and content mods

This project is conceptually similar to KH2’s MOOSE, but tailored specifically to KH1’s engine.

---

## ✨ Features

* 📦 **Memory Expansion**

  * Increases usable memory for game objects and assets

* 🧱 **Object Limit Scaling**

  * Raises limits on loaded entities, models, and effects

* 🎭 **Model & Animation Support**

  * Enables larger and more complex player/enemy models
  * Supports expanded animation data

* ⚙️ **Engine-Level Hooks**

  * Injects into core systems to safely extend behavior

* 🔌 **Mod Compatibility Layer**

  * Designed to work alongside existing KH1 modding tools

## ⚠️ Compatibility

* May conflict with other memory-editing or injection-based mods
* Load order can affect stability
* Not all existing mods are guaranteed to work with GOOSE

If you encounter issues:

* Try running with minimal mods
* Check for conflicts with other frameworks
* Report reproducible bugs (see below)

---

## 🐞 Reporting Issues

If you find a bug or crash, please open an issue and include:

* Description of the problem
* Steps to reproduce
* Game version
* List of active mods
* Crash logs (if available)

---

## 🛠️ Development

GOOSE works by modifying KH1’s runtime behavior through:

* Memory patching
* Hooking internal engine functions
* Expanding static buffers and limits

The goal is to provide a stable foundation for:

* Custom player characters
* Advanced AI behavior
* Expanded asset pipelines
* Large-scale overhaul mods
